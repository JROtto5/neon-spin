# 🎰 NEON SPIN — Idle Slots Empire

A neon/cyberpunk **incremental slot-machine game** that plays start-to-end — spin a single quarter into the Infinite Jackpot across the multiverse. Inspired by Cookie Clicker, built entirely around an animated slot machine.

**▶️ Play locally:** open `index.html` in any browser, or serve the folder:

```bash
python3 -m http.server 8420
# then visit http://localhost:8420/
```

No build step, no dependencies — it's a single self-contained HTML file.

## Features

- 🎰 **Animated 3→5 reel slot machine** with weighted symbols, pairs, full lines, and jackpots
- 🍀 **Luck system** that bends the odds toward rarer, higher-paying symbols
- 🐱 **13 idle generators** (Lucky Cat → Luck Deity) with ×1/×10/×100/MAX buying
- 💰 **30+ upgrades** — payout multipliers, generator boosts, extra reels, faster auto-spin
- 🏆 **30+ achievements**, each granting a permanent +1% to all gains
- ✨ **Golden spins** — click them for Frenzy (×7), Mega Frenzy (×25), Lucky Streaks, and cash drops
- 💎 **Prestige** — cash out for VIP Chips (+2% permanent each) and 7 VIP perk trees
- ⚡ **Auto-spin**, offline earnings, full stats panel
- 💾 **Auto-save** to localStorage with export/import codes
- 🔊 Sound effects, win particles, settings, and an actual ending

## Controls

| Key | Action |
|-----|--------|
| `Space` | Spin |
| `A` | Toggle auto-spin |
| `1`–`6` | Set bet |
| `M` | Max affordable bet |
